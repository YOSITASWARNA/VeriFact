# VeriFact
VeriFact—an AI-based fake news detection system—is a machine learning-powered platform that uses natural language processing designed to identify misinformation and classify articles as real or fake. The system analyzes the textual content of the article and linguistic patterns using deep learning models such as BERT or RoBERTa to detect misinformation and misleading patterns. It aims to help users to verify news authenticity through a web interface before sharing it anywhere on social media, reducing the spread of fake news.
